Source and Binary packages netsim_0.1.0.tar.gz and netsim_0.1.0.zip could be used as packages in R studio. This is the current build (0.1.0) of the code. 
The two functions are gen_graphs and gen_plots. Help file could be accessed via help(gen_graphs) and help(gen_plots) in R command window. 

The complete documentation with interpretation of the plots and how the number of edges are computed will be added shortly.
